<h2 align="center">Getting Started</h2>

1. Clone the repository

    ### `git clone https://github.com/adayarana/vue.git`

2. Install dependencies

    ### `npm install`

3. Set environment variables

4. Run backend

    ### `cd vue/movie-app/backend/`
    ### `npm run dev`

5. Run frontend

    ### `cd vue/movie-app/frontend/`
    ### `npm run serve`

6. Run test

    ### `npm run test` (backend)
    ### `npm test` (frontend)
